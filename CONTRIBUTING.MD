## Commands

### up backend

```shell
> cd back && npm run start
```

### run front dev server

```shell
> cd front && npm run dev
```

### linting backend && front files

```shell
(in front or back folder)> npm run lint
```

### fix backend && front files

```shell
(in front or back folder)> npm run lint:fix
```
