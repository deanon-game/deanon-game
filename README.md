# deanon-game

## add yarn global
```
> npm i yarn -g
```
## install backend dependencies
```
> yarn
```
## install fronted dependencies
```
> cd deanon-front 
deanon-front> yarn
```
## up backend && front
```
> yarn dev
```
## linting backend && front files
```
> yarn lint
```
## fix backend && front files
```
> yarn lint:fix
```
