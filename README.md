# 2gether

Smart chatting with friends!

## dev guide

### add yarn global

```shell
> npm i yarn -g
```

### install fronted & backend dependencies

```shell
> yarn update
```

### up backend && front

```shell
> yarn dev
```

### linting backend && front files

```shell
> yarn lint
```

### fix backend && front files

```shell
> yarn lint:fix
```
